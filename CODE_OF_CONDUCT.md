# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be respectful. Disagreement is fine, disrespect is not.
- Harassment of any kind is not tolerated.
- Report issues to vallabh.joshi@gmail.com with subject `[hush-conduct]`.

Maintainers will investigate all reports and respond within 7 days. Violators
may be removed from the project at the maintainers' discretion.
